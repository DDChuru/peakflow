Screen Guide

Click on numbered badges or hover over UI elements to learn more about each feature.

1

Page Title & Sign In Link

Welcome message with a link for existing users to sign in. Provides clear navigation between signup and login flows.

Technical Details
  • Component: /app/signup/page.tsx
  • Layout: AuthLayout component wrapper
  • Navigation: Link to /login
2

Full Name Input Field

First form field collecting user's full name. Required field with minimum 2 characters validation.

Technical Details
  • Component: Input UI component with tone="dark"
  • Validation: z.string().min(2) via Zod schema
  • Form Library: React Hook Form with register('fullName')
3

Work Email Input

Email address field with mail icon. This becomes the Firebase Authentication username. Validated for proper email format.

Technical Details
  • Validation: z.string().email('Enter a valid email address')
  • Icon: Mail from lucide-react
  • Firebase Auth: Used as primary identifier
4

Password Fields

Two-column grid layout for password and confirm password. Cross-field validation ensures both match. Minimum 6 characters required.

Technical Details
  • Layout: grid grid-cols-1 gap-4 sm:grid-cols-2
  • Validation: z.string().min(6) + .refine() for match
  • Security: type="password" with autoComplete="new-password"
5

Password Strength Tips

Helpful guidance box showing best practices for creating a secure password. Uses glass morphism design with check circle icons.

Technical Details
  • Styling: bg-white/5 border border-white/10 glass effect
  • Content: Static passwordHints array mapped to list
  • Icons: CheckCircle from lucide-react in emerald color
6

Sign Up Buttons

Primary "Create account" button and secondary "Continue with Google" OAuth option. Loading states prevent double submission.

Technical Details
  • Primary Button: Button component with loading prop
  • Google OAuth: loginWithGoogle() from useAuth hook
  • Firebase Auth: Email/password and Google provider configured
  • Redirect: router.push('/dashboard') after success
7

Terms & Conditions Notice

Legal disclaimer informing users that by signing up they agree to the Terms of Service and Privacy Policy. Standard compliance requirement.

Technical Details
  • Location: AuthLayout footer prop
  • Styling: text-xs text-white/60
  • Compliance: GDPR and legal requirement for user consent
