Screen Guide

Click on numbered badges or hover over UI elements to learn more about each feature.

1

Company Header Section

Main company context card showing logo, name, type, status, and industry template. Provides immediate visibility of which company workspace you're in.

Technical Details
  • Data Source: CompaniesService.getCompanyById(companyId)
  • State: useState<Company | null>
  • Route Param: params.companyId from Next.js dynamic route
2

Company Logo

Company logo or initial-based avatar. Supports uploaded logos from Firebase Storage or generates gradient-based initial placeholder.

Technical Details
  • Property: company.logoUrl
  • Fallback: Gradient div with company.name.charAt(0)
  • Size: h-16 w-16 rounded-xl
3

Company Name

Primary company name displayed prominently in the workspace header.

Technical Details
  • Property: company.name
  • Type: string from Company interface
  • Styling: text-2xl font-bold text-gray-900
4

Company Type Badge

Shows company type: "Corporate Company", "Managed Accounts", or "PeakFlow".

Technical Details
  • Property: company.type: 'corporate' | 'manageAccounts' | 'peakflow'
  • Display Logic: Conditional rendering based on type value
  • Purpose: Multi-tenant company differentiation
5

Active Status Badge

Green badge showing company is active. Would show gray "Inactive" if company.isActive is false.

Technical Details
  • Property: company.isActive: boolean
  • Component: Badge variant="default"
  • Conditional Styling: bg-green-100 vs bg-gray-50
6

Industry Template Indicator

Shows industry-specific chart of accounts template applied to this company with account count (889 accounts for Technology & Software).

Technical Details
  • Source: INDUSTRY_TEMPLATES[company.industry]
  • Module: @/lib/accounting/industry-knowledge-base
  • Data: industryTemplate.name and chartOfAccounts.length
7

Manage Companies Button

Admin-only button to navigate to company management page. Only visible if user has admin or developer role.

Technical Details
  • Conditional: hasRole('admin') && <Button />
  • Route: /companies
  • Access Control: RBAC via useAuth hook
8

Financial Overview Section

Section header for KPI dashboard cards showing key financial metrics.

Technical Details
  • Layout: grid gap-4 md:grid-cols-2 lg:grid-cols-4
  • Data: Currently mock data from kpis array
  • Future: Will connect to real-time financial data services
9

Cash Balance KPI Card

Displays current total cash balance across all bank accounts (R245,230) with trend indicator (+12.5% increase).

Technical Details
  • Mock Value: R245,230 (from mock-data.json)
  • Icon: DollarSign from lucide-react
  • Trend: Green color for positive change
10

Revenue MTD KPI Card

Month-to-date revenue total (R128,450) showing +8.3% growth trend.

Technical Details
  • Mock Value: R128,450 (from mock-data.json kpis.revenueMTD)
  • Icon: TrendingUp from lucide-react
  • Color: Blue theme (bg-blue-50, text-blue-600)
11

Outstanding AR KPI Card

Total accounts receivable outstanding (R40,000) with -5.2% decrease (red indicates money owed to company).

Technical Details
  • Mock Value: R40,000 (from mock-data.json kpis.outstandingAR)
  • Icon: Receipt from lucide-react
  • Color: Purple theme (bg-purple-50, text-purple-600)
12

Pending Reconciliations KPI Card

Number of bank transactions awaiting reconciliation (23 pending, +3 new).

Technical Details
  • Mock Value: 23 (from mock-data.json kpis.pendingReconciliations)
  • Icon: FileCheck from lucide-react
  • Color: Orange theme (bg-orange-50, text-orange-600)
13

Quick Actions Section

Section header for quick navigation cards to main workflows.

Technical Details
  • Layout: grid gap-4 md:grid-cols-2 lg:grid-cols-4
  • Data: quickActions array with title, description, icon, href
  • Interaction: Clickable cards with hover effects
14

Bank Import Action Card (NEW Badge)

Navigate to bank statement import workflow. NEW badge indicates recently added feature.

Technical Details
  • Route: /workspace/{companyId}/bank-import
  • Badge: badge: 'NEW' in quickActions array
  • Icon: FileUp with blue theme
15

Reconciliation Action Card

Navigate to bank reconciliation workspace for matching transactions.

Technical Details
  • Route: /workspace/{companyId}/reconciliation
  • Icon: FileCheck with green theme
  • Feature: Bank statement reconciliation workflow
16

Invoices Action Card

Navigate to invoice management - create, view, and manage customer invoices.

Technical Details
  • Route: /workspace/{companyId}/invoices
  • Icon: Receipt with purple theme
  • Feature: Invoice creation and management workflow
17

Reports Action Card

Navigate to financial reports section for insights, P&L, balance sheet, etc.

Technical Details
  • Route: /workspace/{companyId}/reports
  • Icon: BarChart3 with orange theme
  • Feature: Financial reporting and analytics
18

Recent Activity Placeholder

Empty state showing where recent transactions and activities will appear once data is available.

Technical Details
  • State: Empty state component with Sparkles icon
  • Future: Will display recent invoices, payments, reconciliations
