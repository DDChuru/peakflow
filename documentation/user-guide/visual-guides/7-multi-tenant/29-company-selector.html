Screen Guide

Click on numbered badges or hover over UI elements to learn more about each feature.

1

Page Header

Main companies management page with title, subtitle, and breadcrumb navigation.

Technical Details
  • Component: PageHeader from @/components/ui/navigation
  • Props: title, subtitle, breadcrumbs, actions
  • Route: /companies
2

Company Card

Interactive card displaying company information with hover effects and action buttons.

Technical Details
  • Component: Card component with hover state
  • Data Source: CompaniesService.getAllCompanies()
  • Styling: hover:shadow-md transition-shadow group
3

Company Logo

Company logo or initial-based avatar with gradient background. Supports uploaded logos or generates letter-based placeholders.

Technical Details
  • Conditional: Shows company.logoUrl if exists, else gradient with initial
  • Storage: Firebase Storage for uploaded logos
  • Size: h-10 w-10 rounded-lg
4

Company Type & Domain

Shows company type (Corporate, Managed Accounts, PeakFlow) and domain name.

Technical Details
  • Type: company.type: 'corporate' | 'manageAccounts' | 'peakflow'
  • Domain: company.domain property
  • Display: Conditional rendering based on company type
5

Industry Template Badge

Displays the industry-specific chart of accounts template applied to the company, showing template name and number of accounts.

Technical Details
  • Source: INDUSTRY_TEMPLATES[company.industry]
  • Module: @/lib/accounting/industry-knowledge-base
  • Data: Template name and chartOfAccounts.length
6

Search & Filter Bar

Search companies by name, domain, or email. Filter by type (All, Client, PeakFlow).

Technical Details
  • State: searchTerm and filterType useState hooks
  • Filtering: useMemo to compute filtered companies
  • Service: CompaniesService.getCompaniesByType()
7

Active Status Badge

Shows whether company is active or inactive with color-coded badge.

Technical Details
  • Property: company.isActive: boolean
  • Styling: Conditional bg-emerald-50 vs bg-gray-50
  • Toggle: handleToggleStatus() function
8

Edit Company Button

Navigate to company edit page to update details, settings, and configuration.

Technical Details
  • Route: /companies/{company.id}/edit
  • Component: Next.js Link with Button variant
  • Icon: Edit from lucide-react
9

Add New Company Button

Create a new company (only visible to admins and super admins). Opens company creation form.

Technical Details
  • Route: /companies/new
  • Access Control: ProtectedRoute requiredRoles={['admin', 'developer']}
  • Conditional: Only rendered if canManageCompanies === true
