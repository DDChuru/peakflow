Screen Guide

Click on numbered badges or hover over UI elements to learn more about each feature.

1

Create Quote Button

Opens the Create Quote dialog to generate a new quote for a customer. Allows you to select customer, add line items, set pricing, tax rates, and validity period.

Technical Details
  • Component: Button component
  • Action: Opens isCreateDialogOpen dialog
  • Service: QuoteService.createQuote()
  • Validation: React Hook Form + Zod schema
2

Search Bar

Search quotes by quote number, customer name, or other details. Real-time filtering updates the table as you type.

Technical Details
  • State: searchTerm useState hook
  • Filtering: Client-side filtering on quote data
  • Debouncing: No debouncing - instant search
3

Status Filter

Filter quotes by status: Draft, Sent, Accepted, Rejected, or All. Helps focus on quotes in specific stages of the sales cycle.

Technical Details
  • State: statusFilter - QuoteStatus | 'all'
  • Type: QuoteStatus enum from types
  • Options: all, draft, sent, accepted, rejected
4

Quotes Table

Displays all quotes with key information: quote number, customer, dates, status, and total amount. Sortable columns allow easy organization.

Technical Details
  • Data Source: QuoteService.getQuotes(companyId)
  • Type: Quote[] from accounting types
  • Columns: quoteNumber, customer, dates, status, amount
  • Hover: Row highlighting on hover
5

Status Badges

Color-coded badges indicating quote status: Blue for Sent, Green for Accepted, Gray for Draft, Red for Rejected. Visual cues for quick status recognition.

Technical Details
  • Component: Badge component
  • Statuses: draft, sent, accepted, rejected
  • Colors: Tailwind conditional classes
6

Actions Menu

Dropdown menu for quote actions: View, Edit, Send to Customer, Convert to Invoice, Download PDF, or Delete. Context-aware actions based on quote status.

Technical Details
  • Component: DropdownMenu from Radix UI
  • Actions: View, Edit, Send, Convert, PDF, Delete
  • Convert: Creates invoice from accepted quote
  • PDF: pdfService.generateQuotePDF()
