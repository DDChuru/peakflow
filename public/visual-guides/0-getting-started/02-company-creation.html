Screen Guide

Click on numbered badges to learn about the company creation process and industry templates.

1

Company Type Selector - Client Selected

Visual card selector for company type. Client companies are customer organizations, while PeakFlow companies are internal/partner entities. Selection shows with green gradient and checkmark animation.

Technical Details
  • Component: /app/companies/new/page.tsx
  • State: useState<CompanyType>('client')
  • Types: 'client' | 'peakflow' from @/types/auth
  • Styling: Conditional classes with scale-105 and gradient backgrounds
2

Company Type Card Grid

Two-column responsive grid layout for company type selection. Cards use hover effects and transform animations for better UX.

Technical Details
  • Layout: grid grid-cols-1 md:grid-cols-2 gap-6
  • Animations: transition-all duration-300 transform hover:scale-105
  • Selection: handleTypeSelect() updates state and form value
3

Company Name Input

Primary identifier for the company. This field triggers the smart industry detection algorithm as you type, analyzing the name for industry keywords.

Technical Details
  • Validation: z.string().min(2) required field
  • Watch: watch('name') triggers useEffect for industry analysis
  • Debounced: 500ms timeout before calling analyzeCompany()
  • Service: industryTemplateService.analyzeCompany()
4

VAT/Tax Registration Number

Tax registration identifier for compliance and invoicing. Used in generated invoices and financial reports. Optional but recommended for business entities.

Technical Details
  • Field: vatNumber stored in company document
  • Validation: z.string().optional()
  • Usage: Displayed on invoices, quotes, and tax reports
5

Base Currency Selector

Primary currency for all financial transactions. ZAR selected for South African company. Multi-currency support available for international transactions.

Technical Details
  • Field: baseCurrency in company settings
  • Service: currencyService handles conversions
  • Supported: ZAR, USD, EUR, GBP, and more via ISO 4217
6

Industry Template Dropdown - 13 Industry Options

Select from 13 industry-specific templates. Each template includes pre-configured Chart of Accounts, transaction patterns, and common vendors. The suggested option is highlighted based on company name analysis.

Technical Details
  • Source: INDUSTRY_TEMPLATES from industry-knowledge-base.ts
  • Mapping: Object.entries(INDUSTRY_TEMPLATES).map()
  • Templates: 13 industries with 700-950 accounts each
  • Service: IndustryTemplateService.applyIndustryTemplate()
  • Highlight: Suggested industry gets bg-indigo-50 font-medium class
7

Smart Industry Auto-Suggestion Indicator

AI-powered industry detection analyzes company name and description. Shows confidence percentage (87% in this case) and account count that will be provisioned. Quick "Use Suggestion" button available for high-confidence matches.

Technical Details
  • Algorithm: Keyword matching + NLP analysis
  • State: suggestedIndustry and industryConfidence
  • Threshold: Only shows if confidence > 50%, button appears if > 70%
  • Auto-apply: setValue('industry', suggestedIndustry)
8

Industry Template Preview - "889 GL accounts will be auto-provisioned"

Preview of what will be created when you submit. Technology & Software template includes 889 GL accounts, 45 transaction patterns, and 12 pre-configured vendor categories. All accounts are created automatically when the company is saved.

Technical Details
  • Template: INDUSTRY_TEMPLATES.technology_software
  • Accounts: 889 GL accounts including Assets, Liabilities, Revenue, Expenses
  • Patterns: 45 transaction matching patterns for auto-categorization
  • Vendors: 12 common vendor categories (Cloud, SaaS, Marketing, etc.)
  • Process: Created in Firebase batch write during onSubmit()
  • Toast: Shows progress: "Setting up Chart of Accounts..."
