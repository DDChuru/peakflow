Screen Guide

1

Customer Dropdown

Select customer from debtor list. Loads customer payment terms and contact info.

2

Invoice Date Picker

Date invoice is created. Defaults to today. Used to calculate due date.

3

Payment Terms (days)

Number of days until payment is due. Auto-fills from customer default terms.

4

Currency Selector

Invoice currency. Defaults to company base currency (ZAR).

5

PO Number Field

Optional customer purchase order number for reference and matching.

6

Tax Rate Field (Document-Level)

Document-level tax rate pre-filled with company default. Applied to all line items in invoice.

7

Default GL Account Selector

Pre-populates GL account when adding new line items. Saves time for similar items.

8

Line Items Section

Add multiple invoice line items. Click "+ Add Line Item" to add rows.

9

Add Line Item Button

Opens dialog to add new line with description, quantity, price, and GL account.

10

Line Item Table

Shows Description, Quantity, Unit Price, calculated Amount. Each line requires GL account.

11

Totals Section

Real-time calculation: Subtotal, Tax, Total. Updates as line items change.

12

Create Invoice Button

Validates and creates invoice. Uses InvoiceService.createInvoice().
