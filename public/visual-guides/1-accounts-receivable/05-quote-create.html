Screen Guide

Click numbered badges to learn about creating quotes

1

Customer Selector

Select the customer for this quote from your debtor list. Automatically populates customer details.

2

Quote Date

Date the quote is created. Defaults to today's date.

3

Validity Period

Number of days the quote remains valid. Default is 30 days. Calculates the expiry date automatically.

4

Tax Rate

Pre-filled with company's default VAT percentage from settings. Can be overridden per quote.

5

Line Items Table

Add multiple line items with description, quantity, unit price. Amounts calculated automatically. Click "+ Add Line Item" to add more rows.

6

Totals Summary

Real-time calculation of subtotal, tax, and total amount based on line items and tax rate.

7

Notes Field

Optional notes or comments to include with the quote.

8

Create Quote Button

Validates form and creates the quote. Uses React Hook Form + Zod schema for validation.
